Get Started with ILO

An application for theatre control

To get started using ILO just download the software from one of the links bellow. Or if you're feeling up for a challenge feel free to fork the Github repository and help make ILO even better.

This software is still in alpha so please use it at with the expectation that things might break or not work as expected. At this stage it is not recommended to use the software for any show with a "real" audience. If you find a bug, or have any feedback we'd love to hear about it.
